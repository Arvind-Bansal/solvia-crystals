# Solvia Crystals

A premium modern ecommerce website for crystal bracelets and spiritual jewelry built with Next.js, Tailwind CSS, and Framer Motion.

## ✨ About The Project

Solvia Crystals is a luxury crystal bracelet brand focused on combining spirituality, elegance, and modern aesthetics into a seamless online shopping experience.

This project is designed to provide:
- Premium UI/UX
- Fast performance
- Mobile-first responsive design
- Smooth animations
- SEO-friendly structure
- Scalable ecommerce architecture

---

## 🚀 Tech Stack

### Frontend
- Next.js
- React.js
- Tailwind CSS
- Framer Motion

### Backend / Database
- Firebase or Supabase

### Payments
- Razorpay

### Deployment
- Vercel
